📋 DEPLOYMENT INSTRUCTIONS

Quick Test Locally:
  1. cd backend
  2. pip install -r requirements.txt
  3. python verify_deployment.py    # Should pass all checks
  4. uvicorn main:app --port 10000
  5. Visit: http://localhost:10000/api/health

Deploy on Render:
  1. Build frontend: cd frontend && npm install && npm run build
  2. Push to GitHub: git add . && git push
  3. Create Web Service on Render
  4. Build Command: pip install -r backend/requirements.txt
  5. Start Command: cd backend && uvicorn main:app --host 0.0.0.0 --port $PORT
  6. Deploy and monitor logs

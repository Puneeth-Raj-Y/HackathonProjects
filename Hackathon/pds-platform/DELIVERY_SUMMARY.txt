## 🚦 START GUIDE

### Fastest Way to Run

```powershell
# 1. Terminal 1: Setup
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

# 2. Train model
cd backend/ml/scripts
python train_model.py
python generate_sample_data.py

# 3. Start server
cd ../../
python app.py

# 4. Terminal 2: Test
curl http://localhost:5000/api/health
```

**Done!** Server running, ready to test.
